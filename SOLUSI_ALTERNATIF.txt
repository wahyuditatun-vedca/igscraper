📝 CARA PAKAI STREAMLIT SIMPLIFIED

STEP 1: Input Data (📝 Input Data)
├─ Buka Instagram post/hashtag di browser biasa
├─ Copy caption text
├─ Paste di form app → Klik "Add Captions"
│
├─ Atau upload CSV (jika sudah punya dari tool lain)
│
└─ Copy comments (format: username: comment)
   → Paste di form → Klik "Add Comments"

STEP 2: Coding (🏷️ Coding)
├─ Lihat comment satu-per-satu
├─ Pilih kategori Habermas:
│  - FAC = Factual (data, statistik)
│  - NOR = Normative (nilai, kebijakan)
│  - EMP = Emotional (cerita pribadi)
│  - DIA = Dialogical (respon argumen)
│  - NON = Non-Communicative (spam, trolling)
├─ Klik save
└─ Lanjut ke comment berikutnya

STEP 3: Analisis (📈 Analisis)
├─ Lihat distribusi kategori (berapa FAC, NOR, dll)
├─ Lihat bar chart
├─ Lihat DRS Score (Deliberative Rationality Score)
└─ Interpretasi hasil

STEP 4: Export (📤 Export)
├─ Download Posts CSV
├─ Download Comments CSV (with habermas_code)
├─ Download Metadata JSON
└─ Pakai untuk analysis lanjutan / thesis
